No Internet connection

If you are unable to connect to the internet, try the following steps to troubleshoot your connection issues:

1.Verify that your modem or Internet Service Provider (ISP) are not experiencing issues with the physical connections

a.Check the physical connection between your device and modem.

b.Check that you are inside your wireless signal range.

c.Check that your computer's network adapter hardware is not failing due to overheating or age when heavily used.

2.Check the connection to your router using Command prompt

a.In the search box of the Windows Start menu, type Command prompt (or cmd), right-click it and select Run as administrator. Click Yes to confirm. At the command prompt, type ipconfig and press Enter. Look for the IP address listed next to Default gateway, for example 10.1.193.1. Write down the address to use in the next step.

If the ping was successful and you see similar result to the examples but you still cannot connect to the internet, there may be a problem with your network adapter driver, modem or ISP.

3.Make sure that IP address and DNS server are set to obtain the addresses automatically

Click Start, type ncpa.cpl in the search bar and press Enter. Right-click the network icon, select Properties and double-click Internet Protocol (TCP/IP). Make sure that Obtain IP address automatically and Obtain DNS server address automatically are selected.

4.Update the network adapter driver

An outdated or incompatible network adapter driver can cause connection problems. For example, if you recently upgraded to Windows 10, it is possible that the current driver was designed for a previous version of Windows.

5.Temporarily turn off firewall

A firewall can sometimes cause internet connection problems. To check if the connection problem is caused by your firewall, turn it off temporarily and then try to visit www.eset.com.

6.Temporarily disable the protection

Real-time protection can sometimes cause internet connection problems. To check if the connection problem is caused by Real-time protection, turn it off in your ESET security product and then try to visit www.eset.com.

7.Uninstall the network adapter driver and restart

If you still have not identified your connection problems, uninstall your network adapter driver and restart your computer to have Windows automatically install the latest driver. Before uninstalling, make sure that you have a backup driver available. You will need to download the driver using a device (computer) that has an internet connection and then save the driver to a USB flash drive that you can use to install the driver on your unconnected PC.
